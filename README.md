#

Help the little magician restore order to a chaotic workshop. 

Move carefully, place each fragile potion in a proper spot, and avoid breaking any, they are a finite quantity!

### Instructions
- Use WASD or Arrows keys to move
- When pushing a potion, all potions will move in the same direction unless pushed against a wall
- Each potion can be pushed against a wall for a finite amount of times, after which it will be destroyed
- Cover all the targets with potions to complete the level
- Press Enter to start or to reset the current level
